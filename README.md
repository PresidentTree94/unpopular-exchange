# Unpopular Exchange
A NextJS application for sharing, voting, and debating unpopular opinions and pet peeves. It is deployed on Vercel and uses Supabase as a backend. Currently, user authentication has not yet been implemented, so users can vote as many times as they like on takes. Sorting buttons and tabs are also unfinished.
